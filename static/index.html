finalurl

Introduction

finalurl returns redirected urls. If input url doesn't redirect, finalurl returns input url. finalurl allows XmlHttpRequest from anywhere. You can access finalurl with cross domain XmlHttpRequest. This is useful when you want to get redirected url with JavaScript. This is also useful for expanding short urls.

Examples

  • http://finalurl.appspot.com/api?url=http://bit.ly/fAB8Pe
  • http://finalurl.appspot.com/api?url=http://www.google.co.jp/
